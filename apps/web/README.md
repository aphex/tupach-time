# Tupach Time - Shift The Epoch

Shift the epoch to Tupac's exact time of death. Delay the Year 2038 overflow problem by 26 years and turn exact milliseconds into legendary Makaveli ciphers.

## Features

- **Tupac Epoch**: Time starts at September 13, 1996, 4:03 AM (PDT)
- **Year 2038 Safe**: 32-bit overflow delayed until 2064
- **Makaveli Ciphers**: Exact milliseconds converted to Tupac quotes
- **Cloudflare Pages**: Blazing fast global CDN deployment
- **Dark Mode**: Built-in theme support

## Tech Stack

- **Vue 3**: Progressive JavaScript framework
- **TypeScript**: Type safety for JavaScript
- **Vite**: Next-generation frontend tooling
- **Tailwind CSS**: Utility-first CSS framework
- **Cloudflare Pages**: Serverless deployment platform

## Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Deploy to Cloudflare Pages
pnpm deploy
```
